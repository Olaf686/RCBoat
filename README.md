## FAQ

*What components did you use?*
  
For the controller:
  - Arduino uno r3
  - 74HC4067DB multiplexer
  - nRF24l01+ PA+LNA module
  - SSD1306 display
  - Random buttons from ebay
  - 2000 mAh USB powerbank
  
For the boat:
  - Arudino Mega 2560 r3
  - GY-NEO6MV2 gps
  - QMC5883L compass
  - nRF24l01+ PA+LNA module
  - L298N motor driver
  - 12V 3.4Ah lead acid battery
